## 🎯 ما هو Poster Pro؟

**Poster Pro** هو موقع ويب متكامل لإدارة النشر التلقائي على مجموعات Facebook. يوفر:

✅ إدارة حسابات متعددة
✅ إنشاء حملات نشر متقدمة
✅ لوحة تحكم شاملة
✅ جدولة المنشورات
✅ تعليقات وروابط

---

## 🚀 البدء السريع (5 دقائق)

### الخطوة 1: النشر على Vercel (الأسهل)

#### الطريقة الأولى: عبر الويب
1. اذهب إلى: **https://vercel.com**
2. اضغط **"Sign Up"** وسجل دخول بـ GitHub
3. اضغط **"Import Project"**
4. أدخل رابط المستودع:
   ```
   https://github.com/my-firset/poster-pro-website
   ```
5. اختر الإعدادات:
   - **Framework**: React
   - **Build Command**: `npm run build`
   - **Output Directory**: `build`
6. اضغط **"Deploy"**
7. انتظر 2-3 دقائق
8. ستحصل على رابط مثل: `https://poster-pro-website.vercel.app`

#### الطريقة الثانية: عبر Terminal
```bash
# 1. تثبيت Vercel CLI
npm install -g vercel

# 2. تسجيل الدخول
vercel login

# 3. الذهاب إلى مجلد المشروع
cd poster-pro-website

# 4. النشر
vercel --prod

# 5. اتبع التعليمات واضغط Enter
```

## 🎨 الصفحات والميزات

### 1️⃣ الصفحة الرئيسية (Home)
**الرابط**: `/`

**المحتوى**:
- عرض المميزات الرئيسية
- نموذج تسجيل الدخول
- معلومات عن التطبيق

**الميزات**:
- ✓ تسجيل دخول آمن
- ✓ واجهة جميلة
- ✓ معلومات واضحة

---

### 2️⃣ لوحة التحكم (Dashboard)
**الرابط**: `/dashboard`

**المحتوى**:
- إحصائيات المنشورات
- سجل المنشورات
- حالة النشر

**الإحصائيات**:
- 📊 إجمالي المنشورات
- ✅ المنشورات الناجحة
- ❌ المنشورات الفاشلة
- ⏳ المنشورات قيد الانتظار

---

### 3️⃣ إنشاء حملة (Campaign)
**الرابط**: `/campaign`

**الحقول**:
- عنوان الحملة
- محتوى المنشور
- تعليق (اختياري)
- رابط (اختياري)
- تاريخ ووقت النشر

**الميزات**:
- ✓ جدولة المنشورات
- ✓ إضافة تعليقات
- ✓ مشاركة روابط
- ✓ معاينة فورية

---

### 4️⃣ إدارة الحسابات (Accounts)
**الرابط**: `/accounts`

**الوظائف**:
- إضافة حساب جديد
- حذف حساب
- عرض حالة الحساب
- اختيار نوع الحساب

**أنواع الحسابات**:
- 👤 حساب شخصي
- 📄 صفحة
- 👥 مجموعة

## 🆘 استكشاف الأخطاء

### المشكلة: الموقع لا يحمل
**الحل**:
1. تحقق من اتصال الإنترنت
2. امسح ذاكرة التخزين المؤقتة (Ctrl+Shift+Delete)
3. جرب في متصفح آخر

### المشكلة: الصفحات فارغة
**الحل**:
1. افتح Developer Tools (F12)
2. انظر إلى Console للأخطاء
3. تحقق من سجلات Vercel

### المشكلة: الأزرار لا تعمل
**الحل**:
1. تحقق من اتصالك بالإنترنت
2. أعد تحميل الصفحة
3. امسح ذاكرة التخزين المؤقتة
