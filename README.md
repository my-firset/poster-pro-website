# Poster Pro - ناشر برو

موقع ويب لإدارة النشر التلقائي على مجموعات Facebook.

## المميزات

- 📱 واجهة حديثة وسهلة الاستخدام
- 🔐 إدارة آمنة للحسابات
- 📊 لوحة تحكم شاملة
- 🚀 نشر سريع وفعال
- 📅 جدولة المنشورات

## البدء السريع

\`\`\`bash
npm install
npm start
\`\`\`

## البناء للإنتاج

\`\`\`bash
npm run build
\`\`\`

## النشر على Vercel

\`\`\`bash
vercel
\`\`\`
